# 财务对账助手

一个面向财务、出纳、小商家和日常对账场景的小工具，用来减少重复核对数据、检查差异和整理流水的时间。

## 这个工具解决什么问题
- 银行流水、账单、台账之间经常对不上
- 手工比对差异很慢，容易漏看异常数据
- 每周、每月都要重复做对账和汇总
- 不同来源的数据格式不一致，整理成本高

## 核心功能方向
- 流水与台账对账
- 差异检查与异常提示
- 收支明细汇总
- 固定格式财务报表整理
- 多来源表格清洗与统一

## 适合谁使用
- 财务、出纳、对账岗位
- 小公司、工作室、个体经营者
- 电商、门店、培训等有收支流水的业务
- 需要定期核对数据的小团队
